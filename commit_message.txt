Add Playwright e2e criterion tests

This PR implements deterministic Playwright end-to-end tests for the verifiable evaluation criteria of the Prompt Execution Board application.

What this PR does:
- Adds `@playwright/test` devDependency and configures `"test:e2e"` script in `package.json` with `e2e.playwright.config.mjs`.
- Creates `e2e.spec.mjs` containing the canonical region prefix and one Playwright test per testable criterion.
- Automatically captures and asserts failures for unhandled exceptions or console errors via an error listener.
- Includes a test suite covering CRUD operations, Drag and Drop behaviors, WebMCP round-trips via UI observation, visual state verifications, layout/DOM semantics, and simulated execution runs.
- Includes 140 criterion tests spanning all evaluation dimensions, appropriately skipping those that are genuinely subjective.
