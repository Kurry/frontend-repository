# frontend-productivity-backlog-decay-observatory

Backlog decay observatory and priority task manager.

## Judging

This task is judged by an LLM in a headless browser. To test locally:

```bash
# from repository root
uv run corpuscheck test -p tasks/frontend-productivity-backlog-decay-observatory
```

## Running

```bash
# from repository root
uv run corpuscheck start -p tasks/frontend-productivity-backlog-decay-observatory
```
