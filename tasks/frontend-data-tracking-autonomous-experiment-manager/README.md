# Autonomous Experiment Manager

Autonomous Experiment Manager helps a research engineering lead define one objective metric, immutable constraints, a fixed per-run budget, and an incumbent; establish a valid baseline; review AI-proposed hypotheses and inspectable patch hunks; stream one deterministic experiment at a time; compare its measurement with the incumbent under a fixed policy; and atomically keep or revert the patch.

## Judging

To judge this task, open a PR with your implementation in `tasks/frontend-data-tracking-autonomous-experiment-manager/solution/app` and a maintainer will run the judge.

## Running

You can serve the reference application using the standard `npm run dev` inside `solution/app`.
