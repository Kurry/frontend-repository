# Conference Speaker Greenroom Recovery Board

Frontend-only Harbor evaluation task for a conference speaker-slot workbench where failed slots move through a recovery board, linked timing summaries, and a live mobile preview before exporting a portable session artifact.

## Judging

The browser judge evaluates the implementation against the thirteen dimension rubrics under `tests/`. The instruction text is authoritative; reference screenshots are illustrative.

## Running

The builder app must provide `npm run verify:build` and `npm start`, with the latter serving on port 3000.
