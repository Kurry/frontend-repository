# Fictional Risograph Pass & Drying-Rack Sequencer

<summary>
Build a print-run workbench for a wholly invented four-pass poster. The user drags one translucent color pass from fourth to second position, watches the composite proof, overlap cells, pass stack, dependency ribbon, rack schedule, readiness ledger, and packet digest update from the same canonical order mutation, cancels once, commits it, records why the revised order is preferred, handles a later settle-factor correction without rewriting the original decision, approves the rebased run, and exports an exact portable evidence packet. Build it using Tailwind CSS 4.3.2.
</summary>

<core_features>
- Starting clean, drag cobalt 4→2, inspect changed-cell/schedule feed-forward, cancel, repeat/confirm, verify 300 cells/six intervals/one event, brush overlap cells, choose proof, compare alternate order, reveal/cancel/confirm correction, advance 40→69→70→130, note/undo/review/approve/export/reset/import; every surface agrees.
</core_features>

<user_flows>
- Complete poster select → reorder preview/cancel → reorder commit → brush/inspect → prefer proof → alternate-order comparison → correction cancel/confirm → boundary clock run → note/selective view undo → checkpoint/review/approve → exact export/import with no precredited work.
</user_flows>
