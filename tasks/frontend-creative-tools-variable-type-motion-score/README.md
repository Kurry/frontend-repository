# Variable Type Motion Score

A frontend-native kinetic typography editor for authoring responsive 12-second variable font title sequences.

## Running this task

To run this task in a local environment:

```bash
harbor run -p tasks/frontend-creative-tools-variable-type-motion-score -a claude-code -m sonnet
```

## Judging this task

```bash
uv run harbor score <trial-or-job-dir> --task tasks/frontend-creative-tools-variable-type-motion-score
```
