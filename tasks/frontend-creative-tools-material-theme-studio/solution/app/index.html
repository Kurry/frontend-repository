Saved Themes

Themes are saved in your browser's localStorage so that they'll persist between visits to this site.

Tutorial

  1. Palette — Expand primary / secondary rows and pick colors. Toggle Light / Dark under Type.
  2. Editor — ThemeOptions update live. Use Copy theme code to export.
  3. Preview — Switch Phone / Tablet / Desktop and sample site tabs.
  4. Components — Browse Material demos themed by your options.
  5. Saved Themes — Persist multiple themes in localStorage.

Former external affordances (docs, GitHub, Google Fonts marketing) stay interactive but do not navigate.
